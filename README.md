# 🚀 Foresee AI

> **Think Ahead. Decide Better.**

Foresee AI is an AI-powered Decision Intelligence Platform that helps users make smarter, evidence-based decisions before taking action. Instead of simply answering questions, Foresee AI evaluates the context, detects hidden risks and cognitive biases, simulates possible outcomes, and generates actionable recommendations.

Built for **Microcraft VibeThon 2026 – Open Innovation Track**.

---

## ✨ Features

- 🧠 AI-powered decision analysis
- 📂 Intelligent decision classification
- ⚠️ Risk assessment with supporting reasoning
- 🎯 Decision Quality Score & Regret Probability
- 🧩 Cognitive bias detection
- 🔮 Future outcome simulation
- 💡 Personalized alternatives and recommendations
- 📋 Actionable decision plan
- 📊 Interactive analytics dashboard
- 📄 Exportable decision reports
- 📚 Decision history and insights

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/<your-username>/foresee-ai.git
```

### Navigate to the project

```bash
cd foresee-ai
```

### Install dependencies

```bash
npm install
```

### Configure environment variables

Create a `.env.local` file:

```env
GEMINI_API_KEY=your_gemini_api_key
```

### Run the development server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```
